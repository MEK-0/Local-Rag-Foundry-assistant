# CNC Service Handbook

## Spindle Overheating (Error Code: CNC-SP-201)

Description

The spindle temperature exceeded the safety threshold of 85°C.

Possible Causes

- Insufficient lubrication
- Cooling fan malfunction
- High spindle load
- Bearing wear

Diagnostic Procedure

1. Measure spindle temperature.
2. Check coolant circulation.
3. Inspect lubrication system.
4. Verify fan RPM.
5. Review spindle load history.

Recommended Solution

Replace spindle bearings if vibration exceeds 0.15 mm/s RMS.

Maximum Safe Temperature

80°C Continuous
90°C Peak
